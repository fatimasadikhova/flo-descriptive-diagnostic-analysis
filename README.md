# FLO Online Sales Data Analysis

## 📊 Project Overview
This project analyzes the FLO online sales dataset with a focus on **data cleaning, descriptive analysis, diagnostic analysis, and data visualization**.  
The main goal is to extract meaningful business insights from raw transactional data and understand customer behavior, sales performance, and underlying patterns.

---

## 🎯 Objectives
- Clean and prepare raw sales data for analysis  
- Perform **descriptive analysis** to summarize key metrics  
- Conduct **diagnostic analysis** to understand trends and issues  
- Visualize data for better interpretation  
- Build a structured data analytics workflow suitable for business insights  

---

## 🧹 Data Cleaning
The dataset was cleaned and prepared through the following steps:
- Handling missing values  
- Removing duplicate records  
- Fixing incorrect data types  
- Standardizing column names and formats  
- Filtering inconsistent or irrelevant data  

---

## 📈 Analysis Performed

### Descriptive Analysis
- Total sales and revenue overview  
- Customer distribution and behavior  
- Product/category performance analysis  
- Purchase frequency analysis  
- Time-based sales trends (daily, monthly, seasonal)  

### Diagnostic Analysis
- Identifying reasons behind sales changes  
- Customer segmentation insights  
- Detection of anomalies and outliers  
- Relationship analysis between variables (price, quantity, revenue, etc.)  

---

## 📊 Data Visualization
The following visualizations were used to support insights:
- Bar charts for category comparisons  
- Line charts for trend analysis  
- Histograms for distributions  
- Boxplots for outlier detection  
- Heatmaps for correlation analysis
